# Next.js + Tailwind CSS Example

1. add .nvmrc 
2. add .npmrc
3. add  
```json
"engines": {
    "node": ">=16.0.0",
    "yarn": ">=1.22.0",
    "npm": "please-use-yarn"
  },
``` 
in 'package.json' file.
4. run `yarn lint` and modify the file '.eslintrc.json'.
5. install prettier and setting the .prettierrc and .prettierignore files
